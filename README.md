## simple chat bot for coffee shop

### Setup

```bash
pip install -r requirements.txt
```

Create `.env` from `.env.example` and fill `GEMINI_KEY` and `TELEGRAM_BOT_TOKEN`.

### Run

```bash
python bot.py
```

### Commands

- `/assistant your request` - ask the AI cafe assistant.
- `/reset` - reset your current assistant session.
- `/menu` - show the cafe menu.
